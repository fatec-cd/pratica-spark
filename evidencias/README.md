# Pasta de Evidências

Esta pasta deve conter todos os screenshots (capturas de tela) das execuções da atividade prática.

## Estrutura esperada

Coloque seus screenshots nesta pasta com os seguintes nomes:

- `screenshot_fork.png` - Fork do repositório no GitHub
- `screenshot_codespaces.png` - GitHub Codespaces aberto
- `screenshot_estrutura.png` - Estrutura do projeto (ls -la)
- `screenshot_data_generator.png` - Execução do gerador de dados
- `screenshot_wordcount.png` - Execução do word count
- `screenshot_schema.png` - Schema do DataFrame
- `screenshot_receita_categoria.png` - Análise de receita por categoria
- `screenshot_top_produtos.png` - Top 10 produtos mais vendidos
- `screenshot_vendas_regiao.png` - Vendas por região
- `screenshot_docker_build.png` - Build da imagem Docker
- `screenshot_docker_images.png` - Listagem de imagens Docker
- `screenshot_docker_run.png` - Execução do container
- `screenshot_docker_compose.png` - Execução com Docker Compose

## Formato dos arquivos

- **Formato**: PNG ou JPG
- **Qualidade**: Resolução legível (mínimo 1280x720)
- **Conteúdo**: Capture toda a saída relevante do comando/execução

## Exemplo de referência no ENTREGA.md

```markdown
![Fork do Repositório](./evidencias/screenshot_fork.png)
```
